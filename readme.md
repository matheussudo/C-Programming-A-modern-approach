## 📌 About this Repository

Working in cybersecurity operations, I realized the absolute necessity of mastering memory management, reverse engineering logic, and low-level execution. I decided to deepen my knowledge in C and started using **"C Programming: A Modern Approach" (2nd Edition) by K. N. King** as my definitive guide.

This repository serves as my personal notebook and code portfolio. It contains my solutions to the book's exercises, programming projects, and technical notes on how the C compiler and hardware interact under the hood.
